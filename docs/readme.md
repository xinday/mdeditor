# mdeditor 使用說明

即時分割視窗 Markdown 編輯器，支援 GitHub Flavored Markdown、語法高亮、Mermaid 圖表，並自動儲存草稿。

## 快速開始

### 快捷鍵
- **Ctrl + N** —— 新建檔案
- **Ctrl + O** —— 開啟檔案
- **Ctrl + S** —— 儲存檔案
- **Ctrl + Shift + S** —— 另存新檔

### 支援功能
- ✅ GitHub Flavored Markdown（表格、刪除線、任務清單）
- ✅ 語法高亮（100+ 程式語言）
- ✅ Mermaid 圖表（流程圖、序列圖、甘特圖等）
- ✅ 自動儲存草稿至本機（重新開啟時自動還原）
- ✅ 匯出 HTML / PDF

## 常見問題

**Q: 我的檔案會被上傳到伺服器嗎？**  
A: 否。所有編輯、預覽、儲存都在本機進行，沒有任何資料會傳到伺服器。

**Q: 我關閉了編輯器，草稿還會保留嗎？**  
A: 是的。編輯中的內容會自動儲存至本機，下次開啟時自動還原。

**Q: 支援哪些匯出格式？**  
A: 支援 HTML 與 PDF。

**Q: Markdown 或圖表出錯怎麼辦？**  
A: 單個錯誤不會影響其他內容。檢查語法後重新編輯。
